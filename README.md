## Script Repository 👋

This repository is a copy of the Archive volume I use to run Kerboscript scripts
on Kerbal Space Program (KSP). Kerboscript is the scripting language used by the Kerbal
Operating System (kOS) autopilot mod.

The Archive volume corresponds to the Ships/Script folder in the KSP installation folder.
This is ugly (it perhaps should have been in the GameData folder) but it is unlikely
the Developers will fix that now.

The scripts may or may not work depending on a lot of factors. The scripts I featured
on one of my YouTube videos are more likely to work.

The repository also contains a lot of garbage and test files, some of it interesting!

Good luck in your script-writing adventure. And if you ask me to fix any bugs or issues
you find in these scripts I will send you a list of my hourly rates ;-)
